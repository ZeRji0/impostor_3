# IMPOSTOR — Versión Cerveza (Banco v3.4)

**Novedad**: entrada de palabras en **Cooperativo** totalmente **privada por jugador**, como al ver el rol.
- Cada jugador pulsa **“Comenzar mi entrada”**, escribe sus palabras y luego **“Ocultar y guardar”**.
- Después pulsa **“Siguiente jugador”** y pasa el móvil.
- Si ya tenías palabras de todos, puedes **“Usar palabras ya guardadas y empezar”**.

Además:
- Las palabras se reutilizan en rondas siguientes **sin repetirse** y excluyendo las del impostor de cada ronda.
- Nombres por defecto `Jugador1..N` si no usas nombres personalizados.
- Fondo azul cian discreto, interfaz móvil, competitivo, copiar/pegar configuración.

## GitHub Pages
Sube `index.html`, `styles.css` y `script.js` a la rama `main`. En **Settings → Pages** activa *Deploy from a branch* (carpeta `/`).

Favicon opcional/no incluido.
